Parvati Premium is a WordPress plugin for Parvati theme from WPKoi Themes.

== Description ==

Parvati-Premium is a WordPress plugin for Parvati theme from WPKoi Themes.
The features works only with Parvati theme.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/parvati-premium` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
